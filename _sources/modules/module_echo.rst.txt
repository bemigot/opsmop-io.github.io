Echo Module
===========

The Echo module shows strings during the application of an OpsMop policy.
Unlike other modules, strings are automatically assumed to be templates.


.. list-table:: Module Parameters
    :header-rows: 1

    * - Name
      - Help

      - Default
    * - msg
